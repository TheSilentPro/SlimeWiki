SlimeBot

Discord bot for retrieving information on Slimefun items.

About

This is a Slimefun addon that allows you to run a bot using your server that can retrieve information on Slimefun related content.

Any installed addons should work with the bot!

You can view a list of commands by typing / (slash) in the guild while the bot is active.

Alternatevly you can also view them in the commands page.

Installation

  1. Download the latest version from the releases page.
  2. Place it in your plugins folder.
  3. Start server.
  4. Open the config.yml and configure the bot.
  5. Restart the server.
  6. Invite the bot using the link printed in the console.
